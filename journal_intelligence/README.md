# AROS Journal Intelligence v3.0

This layer replaces manual journal expansion with bulk ingestion from authoritative registries.

Primary sources:
- SCImago / SJR
- DOAJ
- UGC-CARE
- ABDC
- ABS / AJG
- JCR / WoS where available

Core principle:
Use ISSN/eISSN for deduplication, not journal title.
